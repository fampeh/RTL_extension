 طریقه نصب اکستنشن (برای مرورگر کروم)
 نصب دستی

ابتدا فایل‌های اکستنشن را از گیتاب دریافت کنید.

اگر با git کار می‌کنید:

bash
git clone https://github.com/fampeh/RTL_extension.git
یا فایل Zip را دانلود کرده و در کامپیوتر خود از حالت فشرده خارج کنید.

باز کردن صفحه مدیریت افزونه‌های کروم
آدرس زیر را در نوار آدرس مرورگر کروم وارد کنید:

text
chrome://extensions
فعال کردن حالت توسعه‌دهنده
در بالای صفحه سمت راست، کلید "حالت توسعه‌دهنده" (Developer mode) را روشن کنید.

بارگذاری اکستنشن
روی دکمه "بارگیری بدون بسته" (Load unpacked) کلیک کنید.
سپس پوشه‌ای که فایل‌های اکستنشن (شامل manifest.json) در آن قرار دارد را انتخاب کنید.

تأیید نصب
پس از انتخاب پوشه، اکستنشن شما در لیست افزونه‌ها ظاهر می‌شود. همچنین آیکون آن در نوار ابزار کروم نمایش داده می‌شود.
