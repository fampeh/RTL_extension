# افزونه راست‌چین متن فارسی

این افزونه متن‌های فارسی را در وب خواناتر و راست‌چین می‌کند و تلاش می‌کند ناحیه‌های کدنویسی، ورودی‌ها و محتوای ذاتاً چپ‌به‌راست را دست‌نخورده نگه دارد.

## ✨ امکانات

- **حالت خودکار (Auto):** به‌محض باز شدن صفحه، متن فارسی را تشخیص داده و راست‌چین می‌کند.
- **حالت راست‌چین (RTL):** تمام پاراگراف‌های فارسی را کامل راست‌چین می‌کند.
- **حالت چپ‌چین هوشمند (Smart LTR):** ترتیب کلمات و اعداد فارسی را اصلاح می‌کند، اما پاراگراف را چپ‌چین نگه می‌دارد.
- **حالت پیش‌فرض (Default):** افزونه را غیرفعال می‌کند.
- پشتیبانی از تیترها، لیست‌ها و نقل‌قول‌های انگلیسی در کنار متن فارسی.
- بدون تغییر در کدها، ورودی‌ها و بخش‌های برنامه‌نویسی.
- سبک، سریع و بدون نیاز به هیچ سرویس خارجی.

## 📥 نصب دستی در Google Chrome

### 1. دریافت پروژه

ریپازیتوری را کلون کنید:

```bash
git clone https://github.com/fampeh/RTL_extension.git
```

یا فایل ZIP را دانلود کرده و از حالت فشرده خارج کنید.

### 2. باز کردن صفحه افزونه‌های Chrome

در نوار آدرس Chrome وارد کنید:

```text
chrome://extensions
```

### 3. فعال کردن حالت توسعه‌دهنده

گزینه **Developer mode** را در بالای صفحه روشن کنید.

### 4. بارگذاری افزونه

روی **Load unpacked** کلیک کنید.

### 5. انتخاب پوشه افزونه

پوشه‌ای را انتخاب کنید که فایل **manifest.json** داخل آن قرار دارد.

پس از بارگذاری، افزونه در فهرست افزونه‌های Chrome ظاهر می‌شود و آیکون آن در نوار ابزار مرورگر قابل مشاهده خواهد بود.
