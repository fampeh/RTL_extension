# افزونه راست‌چین‌کن متن فارسی

این افزونه متن‌های فارسی را در وب خواناتر و راست‌چین می‌کند و تلاش می‌کند ناحیه‌های کدنویسی، ورودی‌ها و محتوای ذاتا چپ‌به‌راست را دست‌نخورده نگه دارد.

## نصب دستی در کروم

1. فایل‌های افزونه را از GitHub دریافت کنید:

```bash
git clone https://github.com/fampeh/RTL_extension.git
```

2. اگر فایل ZIP را دانلود کرده‌اید، آن را از حالت فشرده خارج کنید.
3. در نوار آدرس Chrome این مسیر را باز کنید:

```text
chrome://extensions
```

4. گزینه «حالت توسعه‌دهنده» را روشن کنید.
5. روی «بارگیری بدون بسته‌بندی» کلیک کنید.
6. پوشه‌ای را انتخاب کنید که فایل `manifest.json` داخل آن قرار دارد.

پس از بارگذاری، افزونه در فهرست افزونه‌های Chrome و آیکون آن در نوار ابزار نمایش داده می‌شود.
